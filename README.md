# Tripcraft India

Tripcraft India is a modern React travel itinerary planner for exploring tourist destinations across India and generating personalized trip plans.

## Features

- Explore tourist destinations across India
- Search and filter destinations by state, budget, and travel type
- View detailed destination guides with attractions, weather, budget, tips, and transport info
- See day-wise itineraries with timings, visit duration, meals, evening activities, and local travel guidance
- Get nearby restaurant recommendations based on selected destination
- Generate custom AI-style itineraries using user preferences like comfort level, budget, trip duration, food preference, and travel interest
- Responsive and mobile-friendly design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion
- Lucide React
- React Router

## Run Locally

npm install
npm run dev

Description
This project helps users plan realistic India trips by combining destination discovery, travel filters, detailed guides, restaurant suggestions, and personalized itinerary generation in one clean interface.
